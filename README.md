## Framework of The Game
### Idea
- 創造以3個角色為主角、為期3天、完成至少三個事件、以致有3種結局的PM養成小遊戲
     
### Practice
- 以1個角色、1個主要事件、1個隨機事項及1種結結當作MVP執行
